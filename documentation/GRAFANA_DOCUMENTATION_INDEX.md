## üöÄ Quick Start

**TL;DR - Just want to get it working?**

1. Open: http://localhost:3000
2. Login: admin/admin
3. Add MySQL datasource (mysql:3306, staffAttend_data)
4. Create dashboard ‚Üí Add panel ‚Üí Pie Chart
5. Paste SQL from **GRAFANA_SQL_READY_TO_USE.md**
6. Save & get UID
7. Update .env with UID
8. `php artisan config:clear && php artisan cache:clear`
9. Test: http://localhost:8000/login ‚Üí Dashboard

**Time: ~20 minutes**
